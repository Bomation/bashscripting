| :--- | :--- | :--- |

| 2 | 010 | writes |
| 3 | 011 | read / write |
| 4 | 100 | execute |
| 5 | 101 | read / execute |
| 6 | 110 | execute / write |
| 7 | 111 | read / write / execute |
